
- This mod is for Neoforge 26.1.2

- You might want to turn off View Bobbing


This mod messes with:

- Math functions (Sine, Cosine, Rotation calculations, Square root calculations, abs function, ceil function)

- Worldgen functions

- Menu panorama and splashtext rendering

- Grass, foliage and water colors

- Text

- Audio engine, makes sounds come from the wrong directions.

- Textures (entity texture corruption, texture atlas corruption)

- Item rendering in inventory slots

- Main menu logo rendering

- Cloud rendering

- Rotation math

- Vertex calculations

Important:
- Can cause logspam when generating new chunks, rare, but possible.
