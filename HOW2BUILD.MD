Build instructions

If you want to build the mod yourself (to make changes to the mixins for example):
-- Clone the repo into IntelliJ IDEA
-- Make your changes
-- Go to Gradle view
-- execute runClient so you can test your changes
-- execute build to create the mod JAR
-- Go to the build folder and then libs
-- Open the built JAR in 7-zip, drag mod_icon.png and LICENSE into the JAR
-- You can now run the JAR and also redistribute your modification
